# Android Quiz App

A category-based quiz application developed using Java and Android Studio.

## Features
- Multiple quiz categories
- Score tracking system
- Interactive user interface
- Category selection screen
- Different quiz pages for each topic

## Categories
- Science
- Geography
- Literature
- Mathematics
- Music
- Cinema
- History
- Technology
- Current Events
- Art

## Technologies
- Java
- Android Studio
- Gradle

## Project Structure

app/src/main/
- java/
- res/
- AndroidManifest.xml

## Purpose of the Project

This project was developed to improve Android application development and user interface design skills.
